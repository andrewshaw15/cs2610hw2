Git Command Cheat Sheet

by Andrew Shaw

This webpage contains Git commands, with their names, description, and example usage.

Git Commands:

  1. git init - turn current directory into empty Git repository
    2. Does
  2. git add - add files to staging area for Git
    3. this
  3. git commit - record changes made to files to local repository
    4. piece
  4. git status - return current repository state
    5. of
  5. git config - assign Git settings
    6. code
  6. git merge - combine changes from one branch with another branch
    7. work
  7. git branch - determine what branch local repository is on, add new branch, or delete branch
    8. because
  8. git checkout - switch branches
    9. I
  9. git remote - connect local repository with remote repository
    10. don't
  10. git clone - create local working copy of existing remote repository
    11. know
  11. git pull - get latest version of repository
    12. if
  12. git push - send local commits to the remote repository
    13. so
